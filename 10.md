91. What is data warehouse?

A data warehouse stores large analytical data.

92. What is OLTP?

OLTP (Online Transaction Processing) handles daily transactions.

93. What is OLAP?

OLAP (Online Analytical Processing) analyzes large data.

94. What is star schema?

A schema used in data warehouses.

95. What is snowflake schema?

A normalized version of star schema.

96. What is ETL?

ETL means Extract, Transform, Load.

97. What is data migration?

Moving data between databases.

98. What is database security?

Protecting databases from unauthorized access.

99. What is database scalability?

Ability to handle increasing data or users.

100. Why is SQL important?

SQL is essential for data storage, analysis, and backend development.
