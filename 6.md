51. What is NULL?

NULL represents missing or unknown data.

52. What is NOT NULL constraint?

Prevents a column from storing NULL values.

53. What is UNIQUE constraint?

Ensures all values in a column are unique.

54. What is DEFAULT constraint?

Sets a default value for a column.

55. What is CHECK constraint?

Ensures data meets a specific condition.

56. What is AUTO_INCREMENT?

Automatically generates unique values.

57. What is index?

An index improves database query performance.

58. What is view?

A view is a virtual table created from a query.

59. What is stored procedure?

A stored procedure is a precompiled SQL query.

60. What is trigger?

A trigger automatically executes when events occur.
