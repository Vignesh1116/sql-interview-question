1. What is SQL?

SQL (Structured Query Language) is used to store, manage, and retrieve data from relational databases.

2. What is a database?

A database is an organized collection of structured data stored electronically.

3. What is a table?

A table stores data in rows and columns inside a database.

Example structure:

ID	Name	Age
1	John	25
4. What is a row?

A row (record) represents one entry of data in a table.

5. What is a column?

A column (field) represents a specific attribute of data.

Example: name, age, salary.

6. What is a primary key?

A Primary Key uniquely identifies each record in a table.

Properties:

Unique

Not null

7. What is a foreign key?

A Foreign Key links two tables together.

It references the primary key of another table.

8. What is DBMS?

A Database Management System (DBMS) is software used to manage databases.

Examples:

MySQL

PostgreSQL

Oracle Database

9. What is RDBMS?

RDBMS (Relational Database Management System) stores data in related tables.

10. What are SQL commands?

SQL commands are instructions used to interact with databases.

Main types:

DDL

DML

DCL

TCL
