31. What is a JOIN?

A JOIN combines rows from multiple tables.

32. What is INNER JOIN?

INNER JOIN returns matching records from both tables.

33. What is LEFT JOIN?

LEFT JOIN returns all records from the left table.

34. What is RIGHT JOIN?

RIGHT JOIN returns all records from the right table.

35. What is FULL JOIN?

FULL JOIN returns all records from both tables.

36. What is SELF JOIN?

A SELF JOIN joins a table with itself.

37. What is CROSS JOIN?

CROSS JOIN returns all possible combinations of rows.

38. Why are joins used?

Joins are used to retrieve related data from multiple tables.

39. What is join condition?

Join condition defines how tables are connected.

40. What is join key?

Join key is the column used to link two tables.
