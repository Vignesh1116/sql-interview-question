81. What is backup?

Backup creates a copy of database data.

82. What is restore?

Restore recovers data from backup.

83. What is database schema?

Schema defines database structure.

84. What is metadata?

Metadata describes data about database objects.

85. What is temporary table?

A temporary table exists only during the session.

86. What is indexing strategy?

A method to choose indexes for performance optimization.

87. What is query execution plan?

It shows how a database executes a query.

88. What is clustered index?

Stores table records in sorted order.

89. What is non-clustered index?

Stores index separately from the table.

90. What is database partitioning?

Partitioning splits large tables into smaller parts.
