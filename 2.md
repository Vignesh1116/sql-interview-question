11. What is DDL?

DDL (Data Definition Language) defines database structure.

Examples:

CREATE

ALTER

DROP

12. What is DML?

DML (Data Manipulation Language) modifies data.

Examples:

INSERT

UPDATE

DELETE

13. What is DCL?

DCL (Data Control Language) manages permissions.

Examples:

GRANT

REVOKE

14. What is TCL?

TCL (Transaction Control Language) manages transactions.

Examples:

COMMIT

ROLLBACK

15. What does CREATE command do?

The CREATE command creates database objects like tables.

Example concept:

CREATE TABLE users (id INT, name VARCHAR(50));

16. What does INSERT do?

The INSERT command adds new records to a table.

17. What does UPDATE do?

The UPDATE command modifies existing records.

18. What does DELETE do?

The DELETE command removes records from a table.

19. What does DROP do?

The DROP command deletes a table or database permanently.

20. What does ALTER do?

The ALTER command modifies an existing table structure.
