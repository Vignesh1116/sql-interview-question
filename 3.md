21. What is SELECT?

The SELECT command retrieves data from a table.

22. What is WHERE clause?

The WHERE clause filters records based on conditions.

23. What is ORDER BY?

ORDER BY sorts query results.

Example:

Ascending or descending order.

24. What is GROUP BY?

GROUP BY groups rows with similar values.

Used with aggregate functions.

25. What is HAVING clause?

HAVING filters grouped data.

26. What is DISTINCT?

DISTINCT removes duplicate values from results.

27. What is LIMIT?

LIMIT restricts the number of rows returned.

28. What is LIKE operator?

LIKE searches patterns in strings.

Example:

LIKE 'A%'

29. What is BETWEEN operator?

BETWEEN selects values within a range.

30. What is IN operator?

IN checks if a value matches any value in a list.
