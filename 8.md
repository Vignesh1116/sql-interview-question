71. What is subquery?

A subquery is a query inside another query.

72. What is correlated subquery?

A subquery that depends on the outer query.

73. What is CTE?

CTE (Common Table Expression) is a temporary result set.

74. What is window function?

A function that performs calculations across rows.

75. What is ranking function?

Functions like ROW_NUMBER, RANK, DENSE_RANK.

76. What is partition in SQL?

Partition divides data into logical groups.

77. What is database indexing?

Indexing improves search speed.

78. What is query optimization?

Improving SQL queries for better performance.

79. What is sharding?

Sharding splits a database across multiple servers.

80. What is replication?

Replication copies database data to multiple servers.
